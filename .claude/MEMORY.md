# 题目抽查系统 - 项目记忆

## 项目类型
Electron + Vue 3 桌面应用，用于题库管理和快速查询

## 核心技术
- Electron (主进程)
- Vue 3 + TypeScript (渲染进程)
- Vite (构建工具)
- SQLite/better-sqlite3 (本地存储)
- electron-builder (打包工具)

## 开发环境
- Windows
- Node.js 18+

## 关键文件
- CLAUDE.md - 项目主文档
- .claude/skills/init-project.md - 项目初始化技能
- .claude/skills/run-app.md - 运行验证技能

## 功能模块
1. 题库 CRUD - 增删改查题目
2. 快速搜索 - 关键字匹配，相关性排序
